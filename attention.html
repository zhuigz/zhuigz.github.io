The Loop
一个真正的生存闭环

这不只是"保护注意力"那么简单。它是一个从底层能量到外部价值的闭环：身体能量决定注意力质量，注意力质量决定表达与行动，表达与行动沉淀为作品与产品，作品让你嵌入更贵的社会网络，更贵的网络又反哺你的信息密度与判断力。反过来，能量低、注意力被切碎的人，看似收到很多信息，却没有任何资产沉淀。

Compounding
复利的本质：使用，而不是知道

认知本身并不重要，重要的是使用。一个道理听完的那一刻，并没有真正属于你。只有当你在未来许多天、许多年，甚至在状态最差、最想做糟糕决定的时候，仍然能把它调出来，它才开始变成复利。

LV.1
有印象

听到过，觉得好像有道理。模糊痕迹，无法复述。

LV.2
听过

反复听过七八遍，比有印象更熟，但仍停留在"我知道"。

LV.3
了解

能用自己的话清晰复述，并写出较完整的解释。

LV.4
知道

已经在现实里做到过 —— 做到才算知道。

LV.5
掌握

人生低谷、最想自我攻击时，仍能调用这个方法。

复利不是"我今天懂了一个道理"，
而是这个道理未来每天都能帮我
少浪费一点注意力，多沉淀一点资产。
Attention Debt
注意力负债：你以为在娱乐，其实在被刷

愤怒、短视频、虐恋、比较、抱怨、消费贷 —— 都是不同形态的注意力负债。它们不只消耗当下，还拖累未来。判断标准很简单：当一个东西让你"想离开却离不开"，结束后更空、更乱、更虚弱，它就不再是娱乐，而是在劫持你的注意力。

01
愤怒：把最好的注意力交给讨厌的人

你越恨他、越关注他、越在脑子里反驳他，他在你的精神世界里就越重要。最锋利的反击不是回击，是无视。

02
短视频：不是你在刷它，是它在刷你

以几十秒为单位打断注意力，靠音乐反复调度多巴胺。情绪起落之后是虚无，虚无之后是DMN自我攻击。长期如此会失去完成长任务的能力。

03
虐恋与上头：急切会破坏长期关系

低能量会放大需求感，让人在夜里做出"当下要结果"的不可撤销动作。好的关系让人打开，坏的关系让人收缩。

04
消费贷：透支未来的注意力

欠钱带来对确定性的极度渴望，每天担心如何还。注意力碎片化 → 产出下降 → 收入焦虑 → 借贷消费 → 负债吞噬注意力。

05
比较与意义追问

低能量的DMN漫游里，三四个小时什么也没发生，只是老了三四个小时。

Pressure Formula
深度注意力的压强公式

焦虑往往来自目标太大、时间线太长、未来可能性太多。把目标拆到今天能做的一小步，注意力越聚焦，穿透力越强 —— 高跟鞋能在木板上踩出坑，因为力量集中在很小的接触面积上。

P = F / S

F = 你拥有的注意力与能量　·　S = 你聚焦的面积
面积越小，压强越大；不要追十个方向，把一个方向做到能产生外部结果。

Energy Base
身体能量是一切的底盘

很多人赚不到钱，不只是认知问题，而是身体能量问题。能量低，前额叶难以稳定工作，行动力、判断力、表达力下降；杏仁核更容易上线，恐惧、焦虑、冲动会变得更强。赚钱、表达、做IP、用AI之前，先把身体系统养起来。

🌙

睡眠

不要接触让你睡不着的人、事、念头。为了急一两年牺牲几十年的身体，极不复利。

🍚

血糖

饮食混乱、焦虑性进食、血糖飙升又下降，会让前额叶下线，杏仁核接管。

🔥

炎症

皮肤、肠胃、口腔的慢性问题会持续占用大量能量，让思考、表达都变难。

🔋

ATP

人是"烧炭的机器"，糖、脂肪、蛋白质转化为支撑大脑的能量。没有能量，再好的认知也无法落地。

🦁

向外的生命力

能量充足时，探索欲、表达欲、攻击性更在线 —— 敢公开、敢表达、敢要结果。

🏋️

运动

深蹲、跑步既打断惯性，又重新唤起向外行动的状态。

※ 健康部分仅作为状态管理启发整理，不替代专业医疗建议。

Default Mode Network
DMN：让漫游变成复盘，而不是攻击

DMN是大脑的漫游/发散网络。灵感、复盘、写作都来自它，但能量低时，它会变成自我攻击系统：意义追问 → 社会比较 → 自我批判 → 反复后悔。关键不是"不要想"，而是"不要无产出地想"。

❓

意义追问

负向：我是不是没希望了？
正向：我现在真正的问题是什么？下一步做什么？

⚖️

社会比较

负向：别人都比我强，我太差了。
正向：他强在哪？我能学哪个动作？

🪞

自我批判

负向：我很烂，我不配。
正向：这件事的损失是什么？下次怎么规避？

📝

情绪漫游

负向：刷视频、翻聊天记录、反复后悔。
正向：写下来，做成清单或行动计划。

The AI Era
AI时代：中等智能正在变得便宜

AI最可怕的不是它会聊天，而是它开始稳定输出中高水平智能。文书、设计、代码、营销 —— 大量"中等智能"工作会被压缩。智能信息差迅速缩小，重要的不再是"知道几个工具"，而是：你能不能把AI嵌入真实产出？能不能用它做出作品、产品、商业闭环？

📉

智能过剩

普通文本、资料整理、基础设计、基础代码不再稀缺。需要上移到判断、审美、策略与真实经验。

🎨

表达门槛降低

更多人能写、能画、能剪。空心AI内容会被淘汰，真实经历 + 深度结构才能拉开差距。

🛠️

不要只学工具

学某个按钮容易过时。底层能力是"如何调度AI解决真实问题"。

🎯

把AI绑定到产出

自媒体、文章、视频、产品、咨询 —— 让AI对话变成作品和现实结果，而不是停留在聊得很爽。

🧭

历史视角

农业过剩→粮食便宜，工业过剩→商品便宜，信息过剩→平台争夺注意力，智能过剩→普通智能便宜。

🧠

真正稀缺的

判断、审美、结构、真实产出。一个强大脑持续调度AI，远胜于让AI替你思考。

Business · Expression · IP
公开表达，把黑箱变成资产

很多人觉得自己很厉害却赚不到钱，因为最厉害的东西都留在脑子里、备忘录里 —— 没有公开给世界定价。公开不是炫耀，是让黑箱被看见。被看见才可能被信任、被关注、被购买。

经历 → 封装 → 公开 → 被看见 → 被定价 → 复利

这是表达的核心公式。员工、主管、创业者都需要它 —— 不会表达，能力很难被定价。

📡

IP ≠ 当网红

IP是表达杠杆：把一份注意力复制给成千上万人。一篇文章、一个视频、一份课程都是"注意力结晶"。

💰

商业链路

发现需求 → 做出产品 → 表达营销降低决策成本 → 完成交换。营销本身中性，垃圾交付才可耻。

⏳

结算周期

日结 / 月薪 / 季度 / 年度 —— 收入与你承担的不确定性、调度的资源、影响的人数成正比。

🤝

陌生人交易

熟人关系不好变现，钱与情感混在一起两边难受。陌生人交易反而边界清晰。世界由陌生人组成。

Network & Worldview
你服务什么网络，就获得什么价格

同样是洗碗，在县城、杭州、东京、纽约价格不同 —— 不是劳动不同，是网络不同。先做出成果，让外部看见结果；结果带来信用，信用让更贵的网络愿意接纳你。低价值网络最危险的地方，是让人在自我确认中停滞。

世界观 → 思维 → 判断 → 决定 → 行动 → 习惯 →
结果 → 社会网络 → 反过来塑造世界观

格物致知

🤍

诚意正心

说的、做的、想的尽量一致。长期失信，注意力会被债务感占用，无法稳定向外。

🌊

语言不能替代经验

听一万小时游泳课，不下水仍然不会游。必须下水，必须行动。

🏛️

叙事 + 结构

只有叙事会喊口号；有结构才能改变世界。先理解规则，再利用规则，最后才谈改变规则。

Attention
保护注意力

每天设一段无短视频、无音乐、无社交媒体的深度时间，恢复长注意力链。

Body
恢复身体能量

优先保证睡眠；规律饮食；减少明显让你状态波动的食物；保持运动。

DMN

Capsule

Public

AI

Network
